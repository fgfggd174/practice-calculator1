# Калькулятор площади и периметра

## Описание
Приложение для расчёта площади и периметра геометрических фигур с выбором единиц измерения (сантиметры или метры). История вычислений сохраняется в SQLite.

## Функциональность
- Выбор фигуры: прямоугольник, квадрат, круг, треугольник, параллелограмм, трапеция, ромб.
- Выбор единиц измерения: см или м.
- Ввод параметров с валидацией (положительные числа, неравенство треугольника).
- Расчёт площади и периметра с указанием единиц.
- Сохранение истории в БД с единицами измерения.
- Просмотр, удаление записей, очистка всей истории.
- Стилизация интерфейса (QSS) с переключением светлой/тёмной темы.
- Экспорт/импорт истории в CSV и JSON.
- Сохранение настроек окна (размер, положение, выбранная фигура, единицы, тема) через QSettings.
- Горячие клавиши.

## Горячие клавиши
- `Ctrl+Enter` – рассчитать
- `Ctrl+Shift+C` – очистить поля
- `Del` – удалить выбранную запись из истории
- `Ctrl+Q` – выход

## Установка и запуск
1. Клонировать репозиторий.
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Установить зависимости: `pip install -r requirements.txt`
5. Запустить: `python main.py`

## Структура проекта
- `main.py` – точка входа.
- `ui_main.py` – главное окно (интерфейс + логика).
- `database.py` – работа с SQLite.
- `calc.log` – лог-файл.
- `requirements.txt` – зависимости.
